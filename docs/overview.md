# プロジェクト概要書：カラーパレット生成ツール

| 項目 | 内容 |
|------|------|
| プロジェクト名 | カラーパレット生成ツール |
| バージョン | v1.1 |
| 対象デバイス | PCブラウザ・スマートフォン（両対応） |

---

## 1. アプリの概要

キーワードやベースカラーを入力すると、AIが5色のカラーパレットを自動生成してくれるWebツール。生成したパレットは保存・履歴として残せる。デザイン作業の初期フェーズで色の方向性を素早く探るために使用する。PC・スマートフォンの両方から利用可能。

---

## 2. ターゲットユーザー

- Webデザイナー・UIデザイナー
- 色の方向性を素早く確認したいデザイン初学者
- 作成者自身（開発練習も兼ねた実用ツール）

---

## 3. 主要機能

| # | 機能名 | 優先度 |
|---|--------|--------|
| 1 | キーワード入力によるパレット生成 | Must |
| 2 | ベースカラー指定によるパレット展開 | Must |
| 3 | HEXコードのコピー | Must |
| 4 | パレットの保存・履歴 | Must |
| 5 | レスポンシブ対応（PC・スマホ） | Must |

---

## 4. スコープ外（今回は作らないもの）

- 画像アップロードによる色抽出
- ランダム生成
- 画像としてのダウンロード
- ユーザーアカウント・クラウド同期

---
